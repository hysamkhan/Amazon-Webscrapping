# Amazon-Webscrapping

This repository contains Jupyter notebooks for Amazon web scraping practice and analysis.

## Files

- `AMAZON WEBSCRAPPING.ipynb` - Main scraping notebook.
- `INSIGHTS AMAZON.ipynb` - Analysis and insights notebook.
- `PRACTICE AMAZON.ipynb` - Practice notebook.

## Requirements

- Python 3.x
- Jupyter Notebook or JupyterLab

## How to Run

1. Open this repository in Jupyter.
2. Open any `.ipynb` file.
3. Run cells in order.
